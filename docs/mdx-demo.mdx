# MDX v3 サンプルページ

MDX v3 の様々な機能をデモンストレーションします。

## インタラクティブなコンポーネント

export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '4px',
      color: '#fff',
      padding: '0.2rem 0.5rem',
    }}>
    {children}
  </span>
);

これは <Highlight color="#25c2a0">緑色</Highlight> と <Highlight color="#1877F2">青色</Highlight> のハイライトです。

## カスタムコンポーネント

export const ButtonDemo = () => {
  const [count, setCount] = React.useState(0);
  return (
    <div style={{
      padding: '20px',
      border: '2px solid var(--ifm-color-primary)',
      borderRadius: '8px',
      textAlign: 'center'
    }}>
      <p>ボタンがクリックされた回数: <strong>{count}</strong></p>
      <button
        onClick={() => setCount(count + 1)}
        style={{
          backgroundColor: 'var(--ifm-color-primary)',
          color: 'white',
          border: 'none',
          padding: '10px 20px',
          borderRadius: '4px',
          cursor: 'pointer',
          fontSize: '16px'
        }}>
        クリック
      </button>
    </div>
  );
};

## アラートボックス

:::tip ヒント
MDX では JSX と Markdown を組み合わせて使用できます。
:::

:::info 情報
Docusaurus は MDX v3 を標準でサポートしています。
:::

:::warning 注意
カスタムコンポーネントは `export` キーワードで定義します。
:::

:::danger 重要
インタラクティブなコンポーネントでは React の state を使用できます。
:::

## 画像の埋め込み

### 静的画像（static フォルダ）

`static/img/` フォルダに配置した画像は、ルートパスから直接参照できます：

```markdown
![Docusaurus logo](/img/docusaurus.png)
```

![Docusaurus logo](https://docusaurus.io/img/docusaurus.png)

### インライン画像（相対パス）

ドキュメントと同じディレクトリに配置した画像：

```markdown
![説明テキスト](./image.png)
```


### 外部画像（URL）

```markdown
![プレースホルダー](https://via.placeholder.com/400x200?text=Sample+Image)
```

![プレースホルダー](https://via.placeholder.com/400x200?text=Sample+Image)

### JSX での画像埋め込み

export const ImageWithCaption = ({src, alt, caption}) => (
  <figure style={{textAlign: 'center', margin: '2rem 0'}}>
    <img 
      src={src} 
      alt={alt}
      style={{
        maxWidth: '100%',
        border: '1px solid var(--ifm-color-emphasis-300)',
        borderRadius: '8px',
        boxShadow: '0 4px 6px rgba(0,0,0,0.1)'
      }}
    />
    {caption && (
      <figcaption style={{
        marginTop: '0.5rem',
        fontSize: '0.875rem',
        color: 'var(--ifm-color-emphasis-600)'
      }}>
        {caption}
      </figcaption>
    )}
  </figure>
);

## まとめ

MDX v3 では以下が可能です：

- ✨ JSX と Markdown のシームレスな統合
- 🎨 カスタムコンポーネントの定義
- 🔥 React の state/hooks の使用
- 📊 インタラクティブなデータ可視化
- 🚀 高速なビルドとホットリロード
